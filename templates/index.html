Python e Programação Orientada a Objetos

O que é Python?

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.

Vantagens Profissionais

  • Grande demanda no mercado de trabalho
  • Versatilidade (web, dados, IA, automação)
  • Sintaxe simples e legível
  • Comunidade ativa e muitos recursos

O que é POO?

Programação Orientada a Objetos é um paradigma que organiza o código em "objetos" que contêm dados (atributos) e comportamentos (métodos).

Principais vantagens:

  • Reutilização de código (herança)
  • Organização e manutenção facilitada
  • Encapsulamento (proteção de dados)
  • Modelagem mais próxima do mundo real
